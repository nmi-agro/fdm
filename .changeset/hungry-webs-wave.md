Transform Gerrit from one-shot to streaming conversational assistant

**Agent interaction flow**

- New SSE resource route (`/api/gerrit/:farm/:calendar/stream`) handles three phases: `intent` (returns clarifying questions as JSON), `generate` (streams agent events + enriches plan), `follow_up` (streams conversational replies)
- Intent questions are generated by Gemini Flash based on actual farm data (fertilizer inventory, field characteristics, strategies); if no input is needed the intent phase is skipped and generation starts immediately
- PostHog `$ai_generation` events are tracked for all three phases under the same `$ai_trace_id`

**Session state**

- Zustand store (`gerrit-session.ts`) with `sessionStorage` persistence tracks the full conversation: phase, intent questions/answers, thinking steps, plan, and message history
- Intent answers are serialised as Dutch prose and prepended to `additionalContext` before generation

**New UI components**

- `GerritIntentPanel` — multiple-choice clarifying questions with open-text "Anders" option and per-question context explanation
- `ThinkingSteps` — live step list with elapsed timer shown during generation
- `GerritReasoning` — key-decisions panel with collapsible narrative
- `SuggestedFollowUps` — clickable suggestion buttons after plan is ready
- `GerritChat` — post-generation conversational chat panel

**Other**

- Shared server utility `enrichAndComputePlan()` extracted for reuse between SSE route and accept action
- Model selector removed from strategy form; model selection is now controlled via `gerritModels` config in `fdm-agents`
